本聊天程序支持多客户端,只有注册用户才能登录到聊天室。
程序分服务器和客户端两部分。
   
使用方法：

1.服务器端:
运行服务器程序  
java ChatServer
指定监听端口(大于1024),单击“启动服务”按钮即可启动聊天室的服务器程序；

2.客户端:
运行客户端程序  
java ChatClient
输入服务器程序的IP地址和监听的端口号；
如果没有注册过,需先注册(输入正确的服务器IP地址和端口号之后,单击“注册”按钮,输入信息即可)

注册成功后,返回客户端登录窗口,使用正确的用户名和密码登录；
登录成功后,进入聊天程序主窗口,可以看到当前在线的所有用户,此时即可聊天了。
